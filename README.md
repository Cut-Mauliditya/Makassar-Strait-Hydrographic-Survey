# Hydrographic-Survey-and-Data-Analysis-in-Makassar-Strait
This survey was conducted in the Makassar Strait in collaboration with the National Research and Innovation Agency (BRIN) to characterize the region's physical oceanographic structure, including water mass properties and current patterns. CTD and ADCP data were processed using the instruments' embedded software.

I used Surfer and MATLAB primarily for plotting and visualization refinement, as these tools offered more flexibility than the embedded software for adjusting colormaps, contour intervals, and vector scaling. My contribution focused on producing publication-ready figures to illustrate thermocline structure, water mass properties, and current patterns (including the Indonesian Throughflow).
